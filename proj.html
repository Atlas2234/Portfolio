Unity Horror Game

May 6, 2021

A short horror game that I made by myself with the Unity 3D Engine. The name of the game is called "The Halls". The scripts are written in C# and the map was made with probuilder, a Unity asset. All models, sounds, and textures used in the game are free assets taken from the Unity asset store and freesounds.org. The actual gameplay mechanics and scripts are made from scratch by me. You can watch a video of it here or download and play it here.

CSCb63 Website

April 21, 2021

A redesign and remake of a course website from my university using Flask, HTML, CSS, and Javascript. It was an assignment from my "Introduction to Databases and Web Development" course at University of Toronto, and it was coded by me and a partner. Everything in the assignment was done by us. We made the mockup using mockingbird and adobe XD, designed the frontend with HTML, CSS and Javascript, and programmed the backend with Flask. You can find the project on my GitHub here. The original github repository was made by my partner and was privated so I made a separate one.
